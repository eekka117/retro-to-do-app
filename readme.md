# Retro To-Do App (Windows 98 Style)

A simple Windows 98-style To-Do List web application built with HTML, CSS, and JavaScript.

## Features

- Add, edit, and remove tasks
- Mark tasks as completed individually or all at once
- Clear all tasks
- Task counter showing remaining tasks
- Notifications for task actions
- Saving storage using `localStorage`
- Responsive design: works well on all devices and screen sizes
